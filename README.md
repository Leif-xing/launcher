# 快速启动器

基于 Python + CustomTkinter 开发的快速启动器工具，支持分类管理和一键启动系统工具、应用程序及批处理脚本。

## 快速开始

### 安装依赖

```bash
pip install customtkinter pillow
```

### 运行程序

```bash
python launcher/main.py
```

## 配置文件

配置文件位于 `launcher/config.json`，格式如下：

```json
{
  "categories": [
    {
      "name": "系统工具",
      "items": [
        {
          "name": "任务管理器",
          "icon": "icons/default.png",
          "path": "taskmgr.exe",
          "workdir": ""
        }
      ]
    }
  ]
}
```

### 配置项说明

- `name`: 启动项名称（必填）
- `icon`: 图标路径，相对或绝对路径（可选，默认使用 default.png）
- `path`: 程序路径或命令（必填）
- `workdir`: 工作目录（可选，为空时使用程序所在目录）

## 使用说明

### 基本操作

1. **启动程序**：点击卡片即可启动对应的程序
2. **折叠分类**：点击分类标题右侧的 ▼/▶ 按钮
3. **关闭程序**：点击任意启动项后，程序会自动关闭

### 右键菜单操作

**卡片右键菜单**：
- **编辑**：修改启动项的名称、图标、路径等信息
- **删除**：删除该启动项（需确认）
- **移动到**：将启动项移动到其他分类

**空白区域右键菜单**：
- **添加启动项**：打开对话框添加新的启动项
- **添加分类**：创建新的分类
- **编辑分类**：重命名或删除已有分类
- **导入配置**：从文件导入配置
- **导出配置**：将配置导出为文件
- **备份管理**：查看和恢复历史备份
- **刷新**：重新加载配置文件

### 对话框操作

**添加/编辑启动项**：
- 填写名称（必填）
- 选择图标（可选，支持 .png/.jpg/.ico 等格式）
- 设置执行路径（必填，支持浏览选择）
- 设置工作目录（可选）
- 选择所属分类

**管理分类**：
- 添加新分类：输入分类名称
- 重命名分类：修改已有分类的名称
- 删除分类：删除分类及其所有启动项（需确认）

## 特色功能

### 自动备份机制
每次保存配置时自动创建备份，保存在 `backups/` 目录，最多保留 10 个最近的备份。

### 配置导入导出
- **导出**：可将当前配置导出为 JSON 文件，方便分享和迁移
- **导入**：从文件导入配置，导入前会自动备份当前配置

### 备份管理
通过备份管理界面可以：
- 查看所有历史备份（时间、大小）
- 一键恢复任意备份
- 备份按时间倒序排列

### 流畅动画
- **窗口启动**：程序启动时平滑淡入
- **卡片交互**：点击缩小、悬停放大
- **菜单显示**：右键菜单淡入效果
- **所有动画**：使用帧率优化，流畅不卡顿

### 性能优化
- **图标缓存**：相同图标只加载一次，减少内存占用
- **快速启动**：优化加载流程，启动速度 < 1 秒
- **低内存占用**：运行时内存 < 50MB
